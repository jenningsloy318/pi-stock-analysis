# GICS Taxonomy — Full 4-Level Classification (2023 Revision)

This reference contains the complete Global Industry Classification Standard (GICS) hierarchy
used for industry screening. The screening workflow uses **Level 4 (Sub-Industry)** as the
atomic classification unit for granular sector rotation and company discovery.

## Hierarchy

| Level | Name | Count | Code Digits | Example |

| 2 | Industry Group | 25 | 4 | 4510 — Software & Services |
| 3 | Industry | 74 | 6 | 451020 — IT Services |
| 4 | Sub-Industry | 163 | 8 | 45102010 — IT Consulting & Other Services |

## Screening Workflow Integration

### Phase 1 Enhancement: Sub-Industry RS Ranking

Instead of ranking only 11 sectors, the enhanced screening:
1. Computes RS for all 11 sector ETFs (Level 1 filter)
2. For top 3-4 sectors, drills into sub-industry ETFs/baskets
3. Ranks sub-industries by RS composite within each top sector
4. Identifies the strongest sub-industry trends

### Phase 2 Enhancement: Sub-Industry Deep Dive

The deep dive now operates at Level 4 (Sub-Industry) granularity:
1. Maps the complete value chain within the sub-industry
2. Identifies all publicly traded companies with that GICS sub-industry code
3. Applies sub-industry-specific KPIs from `references/sector_metrics.md`

### Universe Construction
